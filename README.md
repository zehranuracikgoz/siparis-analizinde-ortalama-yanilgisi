\# Sipariş Analizinde Ortalama Yanılgısı



Bu projede, UCI Machine Learning Repository’de yer alan \*\*Online Retail\*\* veri seti kullanılarak

günlere göre sipariş tutarları analiz edilmiştir.



Analizin amacı, yaygın olarak kullanılan \*\*ortalama sipariş tutarı\*\* metriğinin

tek başına yanıltıcı olabileceğini göstermek ve

medyan, işlem sayısı ve kırpılmış ortalama gibi

alternatif ölçülerle daha sağlıklı bir değerlendirme sunmaktır.



\## Kullanılan Teknolojiler

\- Python

\- pandas

\- matplotlib

\- scipy



\## Analizde Ele Alınan Konular

\- Veri temizleme

\- Günlere göre gruplayarak analiz

\- Ortalama vs medyan karşılaştırması

\- Aykırı değerlerin etkisi

\- Kırpılmış ortalama (trimmed mean)



\## Veri Seti

Online Retail Dataset – UCI Machine Learning Repository
